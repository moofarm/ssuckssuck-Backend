### 커밋 메세지


> 커밋 형식입니다.

```
커밋 타입: 내용 (#number)
```  


### Commit Type
|커밋 메세지|설명|
|------|---|
|:page_facing_up::|파일생성/구조변경|
|:art::|코드의 구조/형태 개선|
|:zap::|성능 개선|
|:bug::|버그 수정|
|:fire::|코드/파일 삭제|
|:sparkles::|새 기능|
|:recycle::|코드 리팩토링|
|:pencil2::|오타 수정|
|:twisted_rightwards_arrows::|Merge or PR|



### 브랜치 규칙
> 기본적으로 GitFlow를 따릅니다.
~~~
Feature Branch
feat/{issue number}-{todo}
~~~
